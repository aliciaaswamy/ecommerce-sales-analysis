# E-Commerce Sales & Revenue Performance Analysis

## Project Overview
End-to-end data analysis project on 100K+ real e-commerce orders using the Olist Brazilian E-Commerce dataset.

## Tools Used
- **Python** — Data cleaning, cohort analysis, visualizations
- **MySQL** — Data storage, SQL JOINs and aggregations
- **Tableau Public** — Interactive KPI dashboard

## Key Findings
- Total Revenue: **R$ 16,008,872**
- Top Category: **bed_bath_table** (R$ 1.7M)
- Dominant Region: **São Paulo** (37.5% of revenue)
- Repeat purchase rate dropped **~23% post Q2 2018**
